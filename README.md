# CAN-J1939-Protocal
这是一个专为嵌入式系统设计的轻量级、跨平台 CAN J1939 协议栈​ 实现。项目基于 SAE J1939​ 标准，全面支持 CAN 2.0B​ 与 CAN FD​ 双模通信，并针对 GD32 系列 MCU​ 进行了深度优化和验证，提供完整的硬件抽象层与驱动支持。

✨ 主要特性
- 双模兼容：完整支持 CAN 2.0B (11/29位标识符)​ 与 CAN FD​ 协议，自动适配或手动配置
- GD32 优化：为 GD32F3/GD32F4/GD32E5 等系列提供原生驱动支持，包含完整 HAL 层接口
- 高效传输：利用 CAN FD 的 64 字节数据场，显著提升多包报文（TP）传输效率
- 协议完整：实现 J1939 核心协议（请求/响应、广播、多包传输、网络管理）
- 参数组处理：提供 PGN 解析、封装与维护工具库
- 轻量可移植：模块化设计，可快速移植到 STM32、NXP、ESP32 等平台
- 丰富示例：提供基于 GD32 官方开发板的完整示例工程，开箱即用
